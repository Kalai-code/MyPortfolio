Data Analysis on

Flatten: COVID-19 Survey Data on Symptoms, Demographics and Mental Health in Canada

Introduction:

This dataset is about pre-clinical COVID-19 data,based on survey responses collected from 294,106 Canadians from March 23rd until July 30th 2020, using a platform developed by Flatten, a Canadian non-profit organization. Downloaded the dataset from physionet for learning purposes.

Objective:

To analyze the impact of COVID-19 on mental health, I followed a structured data analytics approach:

  • Data Understanding & Preparation – Examined the dataset to identify key variables, performed data cleaning to handle missing values and inconsistencies, and transformed the data for better usability.
  • Data Analysis & Processing – Loaded the cleaned data, conducted statistical analysis, and explored patterns in mental health trends across different demographics and time periods.
  • Insights & Visual Storytelling – Generated meaningful insights and created visualizations to effectively narrate the impact of COVID-19 on mental well-being, highlighting key trends and correlations.

Solution:

Data Understanding & Preparation:

Tool Used: Jupyter Notebook

I thoroughly reviewed the information available on PhysioNet to understand the data collection process. Here are some key observations:

  • The dataset consists of three CSV files located in the Flatten_Data folder, corresponding to three different survey schemas (Schema1, Schema2, and Schema3).
  • After downloading and inspecting the data, I found that Schema1 contains 263,639 records, Schema2 has 14,932 records, and Schema3 includes 15,534 records.
  • Additionally, I observed that a few columns were uniquely added to each schema, indicating potential variations in data attributes across surveys.
  • After gaining a clear understanding of the data, I merged all three CSV files into a single dataset to streamline the next step in the data analysis process: Data Cleaning.
  • The code for the above step can be found in my GitHub repository.
